General service API


getObjectId

Returns unique generated hash string. 

    pipTestAccount.getObjectId(10);

    // => {{:: objectId }}

getOne

Returns one word from words array which is token randomize. 

    pipTestGeneral.getOne(['green', 'red', 'blue', 'orange', 'white']);

    // => 
{{:: one }}

getOneWord

Returns one word generated randomize. 

    pipTestAccount.getOneWord();

    // => 
{{:: oneWord }}
