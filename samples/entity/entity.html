Entity service API


getEntity

Returns needed entity (event, goal, message etc). 

    pipTestAccount.getEntity();

    // => {{:: entity }}

setContrib

Sets contributor. 

    pipTestAccount.setContrib();

    // => {{:: contrib }}

getOneArea

Returns entity 'Area'. 

    pipTestAccount.getOneArea();

    // => {{:: oneArea }}

getAreasCollection

Returns collection of 'area' entity. 

    pipTestAccount.getAreasCollection();

    // => {{:: areaCollection }}

getOneGoal

Returns a 'goal' entity. 

    pipTestAccount.getOneGoal();

    // => {{:: goal }}

getGoalsCollection

Returns a collection of 'goal' entity. 

    pipTestAccount.getGoalsCollection();

    // => {{:: goalCollection }}
