Data Set service API


ABCD

Returns the english alphabet in the lowercase. 

    pipTestDataSet.ABCD

    // => {{:: ABCD }}

ABCD_CAPITALIZE

Returns the english capitalized alphabet. 

    pipTestDataSet.ABCD_CAPITALIZE

    // => {{:: ABCD_CAPITALIZE }}

DIGIT

Contains simple digitals. 

    pipTestDataSet.DIGIT

    // => {{:: DIGIT }}

SIGN

Returns punctuation signs. 

    pipTestDataSet.SIGN

    // => {{:: SIGN }}

SETTINGS1

Returns settings asset. 

    pipTestDataSet.SETTINGS1

    // => 
{{:: SETTINGS1 }}

SETTINGS2

Returns settings asset. 

    pipTestDataSet.SETTINGS2

    // => 
{{:: SETTINGS2 }}

EMPTY_USER

Simple profile of a user. 

    pipTestDataSet.EMPTY_USER

    // => 
{{:: EMPTY_USER }}

MANAGER_USER

Profile of a user with manager role. 

    pipTestDataSet.MANAGER_USER

    // => 
{{:: MANAGER_USER }}

TESTER_ACCOUNT

Account of a tester user role. 

    pipTestDataSet.TESTER_ACCOUNT

    // => 
{{:: TESTER_ACCOUNT }}

SAMPLER_ACCOUNT

Account of a sampler user. These credentials can be used to login and enter to states requires authorization. 

    pipTestDataSet.SAMPLER_ACCOUNT

    // => 
{{:: SAMPLER_ACCOUNT }}

SERVER_URL

Server url. 

    pipTestDataSet.SERVER_URL

    // => {{:: SERVER_URL }}
