User-Party service API


getOneUser

Returns profile for a user. 

    pipTestUserParty.getOneUser();

    // => 
{{:: oneUser }}

getSettings

Returns settings for entities. 

    pipTestUserParty.getSettings();

    // => 
{{:: settings }}

getParty

Returns party entity. 

    pipTestUserParty.getParty();

    // => 
{{:: party }}

getConnection

Returns connection entity. 

    pipTestUserParty.getConnection();

    // => 
{{:: connection }}

getPartyAccess

Returns party access. 

    pipTestUserParty.getPartyAccess(2);

    // => 
{{:: partyAccess }}

getSession

Returns session object. 

    pipTestUserParty.getSession();

    // => 
{{:: sessions }}
