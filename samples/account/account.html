Account service API


getServerUrl

This method returns mocked server URL. 

    pipTestAccount.getServerUrl();

    // => {{:: serverUrl }}

getSamplerAccount

This method returns mocked typical user account data. 

    pipTestAccount.getSamplerAccount();

    // => 
{{:: samplerAccount }}

getTesterAccount

This method returns mocked typical user account data with tester role 

    pipTestAccount.getSamplerAccount();

    // => 
{{:: testerAccount }}
